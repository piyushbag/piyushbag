# Piyush Bag

Software Engineer at [Arista Networks](https://www.arista.com), based in Santa Clara, CA.

I work on backend and cloud-native infrastructure — distributed systems, service meshes, and the observability and failure-handling layers that keep them running in production. Lately I've been building AI agents that triage infrastructure incidents the way a senior on-call engineer would, rather than relying on brittle threshold alerts.

M.S. in Computer Science from Santa Clara University.

## What I work on

- **Cloud-native platforms** — Kubernetes, Istio, Envoy, and service mesh internals (xDS, Envoy filters, eBPF-accelerated data paths).
- **Observability & AIOps** — turning raw telemetry into useful signal, and replacing static alerting with LLM-driven root-cause analysis.
- **Infrastructure as code** — Terraform and reproducible, automated platform delivery.
- **Open hardware & EDA** — code-first PCB design and CI/CD for hardware, mostly as a side interest.
